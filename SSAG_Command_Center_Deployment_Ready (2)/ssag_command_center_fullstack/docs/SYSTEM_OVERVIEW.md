# SSAG Command Center - System Overview

## Purpose
This system is the control layer for SSAG. It provides one web-based place to view divisions, deals, clients, finance, alerts, and activity.

## Core workflow
Lead -> Deal -> Closed Deal -> Client -> Invoice -> Payment -> Dashboard visibility

## Modules
- Authentication
- Executive Dashboard
- Division Management
- Deals Pipeline
- Client Accounts
- Finance / Invoice Ledger
- Alerts
- Activity Log
- System Documentation

## Source of truth
This app becomes the operational source of truth once populated with real data.

## Version
v1 foundation full-stack build
